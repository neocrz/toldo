# Toldo
A simple lua todo (task) manager

```console
Usage: toldo [-h] <command> ...

Todo list with lua.

Options:
   -h, --help            Show this help message and exit.

Commands:
   add                   Add a todo.
   list                  Lists all the todos from a category.
   rm                    Remove a todo.
   check                 Check or uncheck a todo.
```
